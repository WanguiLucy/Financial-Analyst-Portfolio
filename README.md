## Projects

### SQL — Kenya Banking Sector Analysis *(in progress)*
- **Data:** CBK Banking Supervision Reports 2019–2023
- **Tools:** PostgreSQL, DBeaver
- **Questions answered:** Which banks carry the highest credit risk? How has industry ROE trended over 5 years? Which tier of banks is most efficient?
- **Folder:** /sql/project-banking

### Excel — Financial Models *(coming soon)*
- **Companies:** Equity Bank, Safaricom, KCB Group
- **Tools:** Microsoft Excel
- **What I will build:** Linked 3-statement models (P&L, Balance Sheet, Cash Flow), DCF valuation, scenario and sensitivity analysis
- **Folder:** /excel-models

### Power BI — Kenya Banking Dashboard *(coming soon)*
- **Data:** CBK + NSE data
- **Tools:** Power BI, DAX
- **What I will build:** Multi-page interactive dashboard covering industry KPIs, bank-by-bank comparison, NPL risk tracker, and trend analysis 2019–2023
- **Folder:** /power-bi

### Python — Credit Risk Model *(coming soon)*
- **Data:** CBK loan portfolio data
- **Tools:** Python, pandas, scikit-learn
- **What I will build:** Logistic regression and random forest model to predict credit default probability, with model accuracy comparison and Power BI dashboard of results
- **Folder:** /python
